# Landing Monteazul & El Sauce

Landing inmobiliaria responsive desarrollada en HTML, CSS y JavaScript puro.

## Cambios de esta versión

- Identidad visual renovada en azul Monteazul.
- Hero de alto impacto con profundidad, iluminación y tarjetas tipo glass.
- Navegación más moderna y menú móvil mejorado.
- Carrusel de proyectos con tarjetas premium, filtros y controles.
- Nueva franja de confianza y beneficios comerciales.
- Secciones de proceso, marcas y contacto rediseñadas.
- Ilustraciones internas recoloreadas para eliminar el predominio verde.
- Formularios conectados con WhatsApp.
- Logo oficial de Monteazul incorporado en header, footer y favicon.

## Logo

El proyecto ahora usa:

- `assets/logo-monteazul.png` → logo principal en la interfaz.
- `assets/favicon.png` → icono del sitio.

Si luego deseas una versión con fondo transparente en mayor calidad o el logo de El Sauce, se puede reemplazar fácilmente manteniendo los mismos nombres de archivo o actualizando las rutas en `index.html`.

## Uso

Abre `index.html` en un navegador. Para publicar, sube toda la carpeta conservando su estructura.

## Validación pendiente

Confirma teléfonos, precios, áreas, nombres de proyectos, condiciones comerciales y fotografías oficiales antes de publicar.
